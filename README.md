Life Hack or Urban Myth App

Overview
Some people wonder if everyday shortcuts really work. The app checks those claims using a quiz you can play. It shows what is fact, what is fiction. One question at a time, answers come clear. Not everything shared online holds up. This tool tests each trick differently. Surprises happen when belief meets proof.

Features
- Welcome screen with start button
- Flashcard quiz system
Some say it works. Others call it nonsense. One claims proof. Another sees only guesses. This version says yes. That one insists no. Try believing what fits. Or doubt everything instead. Truth hides somewhere between
- Score tracking
- Feedback after each answer
Score shown at the end

How It Works
A single flashcard lives inside the app, holding a statement along with a true-or-false tag. One after another, they appear when the user taps a reply. If the pick matches the stored value, points shift upward before advancing forward. After checking, the total adjusts - then the next card slides into view.
